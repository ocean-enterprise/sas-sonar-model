# Synthetic Aperture Sonar (SAS) Detection Range Model

A mathematical model for SAS sonar for the purposes of studying SAS deployed to AUV platforms, and how it compares with magnetic sensing in a variety of conditions.

## Install & Usage
1. Clone
2. Install dependencies using `pip install -r requirements.txt`
3. Run
```sh
python sas_model.py
```
